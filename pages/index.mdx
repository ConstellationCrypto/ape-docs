# What is ApeChain?
ApeChain is a gaming-focused L3 designed to serve as a home base and cornerstone for the Ape community, offering high performance, ultra-low fees, and a dedicated infrastructure layer for upcoming games in the ApeCoin ecosystem.

Through Caldera and Arbitrum's custom native token feature, ApeChain will utilize $APE as its native gas token, providing significant utility to the token while enabling a rich native economy powered by $APE.
